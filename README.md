# Game of Life CLI

Conway's Game of Life implemented in Rust to run in the terminal.

## Try it out

For the default settings:
```shell
cargo run

```
To run with custom settings from files:
```shell
cargo run -- -i <filename>

```

### Example

Try it out with the beacon example in the repository:
```shell
cargo run -- -i beacon

```
